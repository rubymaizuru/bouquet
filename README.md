# Bouquet

[花束問題](http://www.benkyoenkai.org/2015/01/v12.html) を解決するアプリケーションです。

## Database creation

```
$ bin/rails db:create
```

## Database initialization

```
$ bin/rails db:migrate
```

## Usage

```
$ bin/rails s -b 0.0.0.0
```
ブラウザで http://localhost:3000 を開きます。

## How to run the test suite

```
$ bundle exec rspec
```
